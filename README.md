# footnotes-component
Easy Footnotes in HTML Tags

## how to use

```html
  <script src="./foot-notes.js"></script>
  <div>Your text here, and include a footnote
    <foot-note>Foot Note Here</foot-note>
  </div>
  <foot-notes>
    <!-- This div will include all foot notes -->

  </foot-notes>

```
